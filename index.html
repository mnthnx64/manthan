Overview of Project: Reinforcement Learning based image denoiser. Various approaches and algorithms have been used to reduce noise from images to improve their overall clarity. The use of Reinforcement Learning (RL) to achieve this task has, so far, been uncommon. One such method/ framework is PixelRL [1], which treats each and every pixel as a state and learns efficiently. Here, we propose to use a modified PixelRL framework to denoise color images
